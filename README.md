MP3 to WAV Converter
This is a simple web-based MP3 to WAV converter built using Streamlit and PyDub.

Features
File Upload: Upload your MP3 file easily.
Conversion: Converts the uploaded MP3 file to WAV format.
Download: Provides a download link for the converted WAV file.
How to Use
Upload: Select an MP3 file using the file uploader.
Conversion: The app automatically converts the MP3 file to WAV.
Download: Once the conversion is done, download the converted WAV file.
Setup
To run this application locally:

Clone this repository.
Install the required dependencies (streamlit, pydub).
Run the application using streamlit run app.py.
